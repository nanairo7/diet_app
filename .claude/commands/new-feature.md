以下の手順で新機能を実装してください。

1. `main` の最新から `feat/<機能名>` ブランチを新規作成する
2. 実装方針・影響ファイルを整理して報告する（実装前に確認を取る）
3. 承認後、実装する
   - テキスト文字列は `lib/constants/app_strings.dart` に定義する
   - 新画面・ウィジェットには対応するテストを追加する
4. `flutter analyze` → `flutter test` がパスすることを確認する
5. `feat:` プレフィックスでコミットする
6. 影響する設計書（`docs/design/`）を更新する
7. PR を作成して URL を報告する

機能概要: $ARGUMENTS
